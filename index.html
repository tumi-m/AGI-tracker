Install as an app

PHONE · TABLET · DESKTOP

Tiresias is an installable web app. One tap puts the clock on your home screen or dock — full screen, offline, straight from the browser. No store, no account.

On iPhone / iPad: Share → Add to Home Screen. On Android and desktop Chrome or Edge: menu → Install app.

Renders inline and inherits your page's typography. For sites that allow JavaScript.

Data

One JSON file. Fetch the target date once, count down locally — no polling, works offline.

// target dates change rarely
const r = await fetch("https://tiresias.io/data/forecasts.json");
const { forecasts } = await r.json();

Method

The headline number is the median of ten tracked forecast dates — lab CEOs, named researchers, forecasting communities and prediction markets — recomputed from the public dataset on every page load. Anyone can verify it in three lines of code.

Every date links to a primary source and carries a verification stamp. When a forecaster revises, the old date is kept in the dataset's history — revisions are the story, not an embarrassment to hide.

Definitions of AGI differ between sources, and each date is the midpoint of a wide distribution, not a promise. This site takes no position on who is right. The disagreement is the point.
